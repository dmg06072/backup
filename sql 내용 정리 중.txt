딸깍 버전
-유저 계정 & 권한
계정 생성
좌측바 어드미니스트레이션 -> 유저 앤 프리빌리지 -> 애드 어카운트 -> 로그인 -> 로그인 네임, 리밋 투 호스트 매칭 : localhost, 비번 입력하기
권한 부여
스키마 프리빌리지 -> 유저 선택 -> 애드 엔트리... -> 권한 부여할 스키마 선택 -> 아래에 있는 부여할 권한 체크 -> 어플라이

-원격 접속
공용 계정 만들기
create user remote@'%' indentified by '1234';
% = 시작 IP가 어디든 상관 없이
grant 부여할 권한 on 스키마.* to remote@'%';

- csv or json 파일 sql로 불러오기
좌측 네비게이션 바 -> 테이블 우클릭 -> 테이블 데이터 임포트 위자드 -> 파일 링크

- 파일 백업
좌측 네비게이션 바 -> 데이터 익스포트 -> 백업할 파일 선택 -> 옵션 설정 후 스타트 익스포트
-파일 복원
좌측 네비게이션 바 -> 데이터 임포트/리스토어 -> 폴더 선택 -> 스타트 임포트

- cmd 창에서 sql 명령어 사용 안 될 때
sysdm.cpl -> 고급 -> 환경변수 -> path 편집 -> 새로만들기 -> C:\Program Files\MySQL\MySQL Server 8.0\bin -> 저장

- 명령어로 백업하기
백업할 폴더 경로에서 cmd
단일 백업 : mysqldump -u root -p [DB이름] > [파일이름].sql
전체 백업 : mysqldump -u root -p --all-databases > all_backup.sql
- 복원하기
mysql -u root -p [DB이름] < [백업파일].sql
