1. Network 는 무엇이며 무엇을 목적으로 하는가?
데이터 교환을 위해 통신장치를 연결한 통신망

2. Network 의 종류는 크게 ooo/ooo 로 나눌 수 있다
LAN / WAN

3. LAN 의 특징에 대해서 서술하세요
비교적 작은 지리적 범위에서 사용(근거리 통신망)
고속 통신 목적
낮은 설치/운영 비용
간단한 설치와 유지보수
외부보다 보안이 쉬움

4. WAN의 특징에 대해서 서술하세요
넓은 범위를 연결(광대역 / 원거리 통신망)
다양한 전송 기술 사용(다양한 데이터 Type 교환)
안정성(신뢰도) 중점
인터넷 기반 연결 가능 (VPN 포함)
높은 운영 비용
복잡한 네트워크 관리 필요

5. InterNet 이란
전 세계 공용 네트워크. 누구나 접근 가능(LAN + WAN을 묶은 거대 통신망)

6. 프로토콜은 무엇인가?
통신을 위해 사전에 정의한 규칙
통신용 프로그램
Syntext + meaning = timing

네트워크 환경에서 통신을 위해 사용하는 규칙의 집합
데이터 포맷, 전송 순서, 에러 제어 방식 등 포함

7. 네트워킹 모델의 목적은 무엇인가?
기기간 통신 위해서 사용되는 지침서
문제발생시 확인하는 문제해결 가이드
OSI&TCP/IP : 이기종간의 통신을 위해서 제작한 표준 지침서

효율적인 네트워크 설계를 위한 표준 모델
문제 발생 시 원인 파악 용이

8. TCP/IP 모델과 OSI 7 모델의 계층을 적고 각 계층별 역할을 적으세요
-TCP/IP 모델
애플리케이션 계층
전송 계층
인터넷 계층
데이터 링크 계층
물리 계층

-OSI 7 모델
7 애플리케이션(응용) 계층	: 통신을 수반하는 프로그램(사용자/장치에 의해 작업물 생성)
6 프레젠테이션(표현) 계층	: jpg, 작업물 압축, 인코딩, 암호화
5 세셴 계층				: 지속적인 데이터 교환을 위해 사전에 대화형 채널 생성
4 전송 계층				: 송수신지간(End to End) 데이터의 운송방식 결정
3 네트워크 계층			: 경로 탐색
2 데이터링크 계층		: 장치간(Point to Point) 데이터의 운송방식 결정
1 물리 계층				: 장치간 물리적 연결
